# オンライン読書会チャットサービス 要件まとめ v2

## コンセプト

- 顔出しなし・テキストチャットのみ
- 会員登録不要
- 絵チャットルーム的な気軽さ
- ルームは一時的に存在し、自動で消える

---

## 技術スタック

| 項目 | 選択 |
|------|------|
| フロントエンド | React（Next.js） |
| リアルタイム通信 | Supabase Realtime（Broadcast + Presence） |
| DB | roomsテーブルのみ（メッセージはDB保存なし） |
| ホスティング | Vercel |
| 費用 | 無料枠で運用可能（同時接続200まで・1週間無操作で停止） |
| 開発ツール | Claude Code + Supabase MCP |

### Supabase Realtimeの役割
- **Broadcast**：チャットメッセージをリアルタイム配信（DB保存なし、送ったら消える）
- **Presence**：参加者の在席管理 + ルーム一覧表示（DBいらず）

---

## ルーム管理

- **自動削除条件**
  - 参加者が0人になったら即時削除
  - 作成から1.5時間経過で自動削除
- **ルームタイプ**（作成時に選ぶ）
  - 自由参加型：各自が今読んでる本を設定、カジュアルな雰囲気
  - 課題本型：本のタイトル・著者をルーム共通で設定、議論向けUI
- **鍵付きルーム**：パスワードを設定して参加者に別途共有する方式

## UI・デザイン

### 画面レイアウト

```
┌──────────────────────────────────┐
│  ヘッダー（ルーム名・本・人数・モード切替）  │
├──────────────────────────────────┤
│                                  │
│   焚き火エリア（上部 約55%）          │
│   参加者アイコンが焚き火を囲む           │
│   発言すると吹き出しが出る             │
│                                  │
├──────────────────────────────────┤
│  Discord型チャットログ               │
│  （名前色分け・全員左揃え）             │
├──────────────────────────────────┤
│  入力欄 ＋ 送信ボタン                │
└──────────────────────────────────┘
```

### 焚き火エリア
- 中央に焚き火（SVG turbulenceフィルターで本物らしい炎アニメーション）
- 参加者アイコンが焚き火を囲むように楕円配置
- 人数に応じてアイコンが均等に増減
- 背景は暖色系の暗闇、星のドット、地面の光

### 吹き出し
- 発言するとアイコンから吹き出しが出る（約3秒で消える）
- 吹き出しは左右横方向に統一（上下が切れないよう）
  - 右半分のアイコン → 右に出る
  - 左半分のアイコン → 左に出る
- 幅固定（2行以内に収まるサイズ）

### チャットログ
- Discord型（全員左揃え、自分も含む）
- ユーザーごとに名前の色を自動割り当て
- アイコンは絵文字（nano bananaなどで本番素材を作る予定）

### カラーテーマ
- 背景：暗い暖色系（#080608ベース）
- 焚き火の光が全体にほんのり当たるグラデーション
- チャットログは落ち着いた暗め

## 機能一覧

### 必須機能
- [ ] ルーム作成（タイプ選択・タイトル・課題本設定）
- [ ] ルーム一覧表示（Presenceから動的生成）
- [ ] 匿名ユーザー名設定（入室時に決める）
- [ ] テキストチャット（Broadcast）
- [ ] 参加者一覧（Presence）
- [ ] 感想タイム／雑談タイムの切り替えフラグ
- [ ] 参加者0人で自動削除
- [ ] 1.5時間で自動削除
- [ ] 鍵付きルーム（パスワード設定）

### UIモードの違い（自由参加型 vs 課題本型）

| 機能 | 自由参加型 | 課題本型 |
|------|-----------|---------|
| 感想タイム切り替え | オフ（デフォルト） | オン（デフォルト） |
| 本のタイトル表示 | 各自が任意設定 | ルーム共通で設定 |
| UIの雰囲気 | カジュアル | 議論向け |

## デザイン検討の変遷

| 案 | 結果 |
|----|------|
| ニコニコ風コメント流れ＋チャットログ | 人が少ないと上半分が空になる問題 |
| 正円の円卓＋アバター | 吹き出しの配置・サイズ問題が続く |
| 楕円の円卓 | 均等配置はよくなったが方向性に迷い |
| 横一列アバター | 発表会みたいで会話感が出ない |
| **焚き火を囲む楕円配置（現採用）** | 会話感・雰囲気ともに◎ |
